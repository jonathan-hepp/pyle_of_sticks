Pyle of Sticks Game

A self-learning Python-based version of the pile of sticks game.

How To Play

The game starts with a "pile" of 20 sticks
You play against the computer
On its own turn, each player takes up to 3 sticks off the pile
The player that takes the last stick loses
Note: you can also use the number keys to play with the keyboard

Game Info

About the name

No, it's not a typo, it's just a bad pun. At this point, naming stuff developed in Python like that is almost a tradition.

Origins

This game is an implementation of the programming exercise described here.

Implementation details

Every time the computer wins a match, the details of the game are saved in its internal memory. Therefore, the more games the computer wins, more data gets collected for future games, which then makes it a harder adversary.

Technologies involved
